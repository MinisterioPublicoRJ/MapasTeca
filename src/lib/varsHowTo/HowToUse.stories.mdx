# Como aplicar um estilo customizado no seu projeto

Para alterar o valor de uma variável da MapasTeca em um projeto que a utiliza, basta declarar a variável correspondente e o valor desejado em um lugar acima do componente desejado.

### Exemplo:
Utiliza-se o componente "Small Badge" dentro do componente "Section Header".
Para alterar a cor de Small Badge, adicione o seguinte código ao CSS do componente pai em Section Header:

###### --colorDanger: #ababab;

Você pode até usar outra variável específica do seu projeto para os componentes da MapasTeca:

###### --colorDanger: var(--minhaPropriaVariavel);
