# O que é a MapasTeca

A MapasTeca Web é uma biblioteca de componentes independetes organizados em atomic design. Qualquer um desses componentes pode ser utilizado e personalizado para o seu próprio projeto.

## Qual o benefício de utilizar a MapasTeca?

Simplesmente dizendo, escrever menos código. Os componentes são independetes e funcionais e podem salvar alguns minutos extra.

## Mas a paleta da MapasTeca não tem nada a ver com o meu projeto!

Muito simples: Apenas modifique as variáveis de CSS da MapasTeca no seu projeto! Para saber mais sobre isso, cheque a sessão de variáveis e a documentação de cada componente (ela mostra o que nele pode ser alterado).
